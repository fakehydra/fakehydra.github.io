磁盘容量大小=柱面存储容量*磁道数量

[root@jinc ~]# echo 8225280*2610|bc
21467980800
[root@jinc ~]# echo 8225280*2610/1000/1000/1000|bc
21
最终实际大小  21（取整）

利用python计算小数
python
>>> 8225280.0*2610/1000/1000/1000
21.467980799999999~~约等于21.5
这就是我们的实际大小
