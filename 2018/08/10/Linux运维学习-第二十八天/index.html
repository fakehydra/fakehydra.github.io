循环语句说明

变量名称规范：字母、数字、下划线；尽量不要和环境变量名称冲突

三剑客命令之首–awk

数组说明：

用于统计文件信息，出现的数量

awk语法格式

awk [选项] '模式{动作}' file

awk正则匹配说明

说明：awk可以直接识别正则

特殊模式说明

awk 'BEGIN{print 3+1}' 
BEGIN{}  --awk 特殊模式；先执行

说明：awk命令在使用print输出指定信息的时候，自定义字符信息需要用双引号括起来；默认awk会将没有双引号括起来的信息识别为变量；因为在awk中$xx 理解为取相应列

BEGIN作用说明

1.用于测试
2.用于计算
3.修改内置变量

END –结束补充；求和

[root@jinc files]# awk '{i=i+$NF} END{print i}' test_sum.txt 
55

awk指定分隔符

awk -F "[:]"  --以 : 作为分隔符
 awk 'BEGIN{FS=":"}'  --以 : 作为分隔符 
awk -vFS=":" 'BEGIN{FS}'  -- -v 指定，后面调用

课堂练习

统计    /etc/service 空行
方法一：grep
[root@jinc files]# grep '^$' service.txt |wc -l
 4
 [root@jinc files]# grep -c '^$' service.txt 
 4

方法二 ：利用awk（模式{动作}）

01.定义awk的条件信息（模式）
[root@jinc files]# awk '/^$/' service.txt 
02.定义统计数据信息的的动作
i=i+1  <<默认最开始i为0，
[root@jinc files]# awk '/^$/{i=i+1;print i}' service.txt 
1
2
3
4
[root@jinc files]# awk '/^$/{i=i+1}END{print i}' service.txt 
4
[root@jinc files]# awk '/^$/{i++}END{print i}' service.txt 
4

数组说明：

[root@jinc files]# awk 'BEGIN{i[1]="www";i[2]="post";i[3]="mp3"; print i[1];i[2]}'

[root@jinc files]# awk 'BEGIN{i[1]="www";i[2]="post";i[3]="mp3"; print i[1],i[2]}'
post
[root@jinc files]# awk 'BEGIN{i[1]="www";i[2]="post";i[3]="mp3"; print i[1],i[2],i[3]}'
post mp3
