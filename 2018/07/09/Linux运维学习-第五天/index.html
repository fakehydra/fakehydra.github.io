  2018-07-09    学习笔记

简介

第一关测验练习题–续

第一关测验练习题

  1. 把oldboy.txt 拷贝（复制）到/tmp 下

    cp /data/oldboy.txt /tmp
    cp –复制文件或复制目录
    -r —递归复制目录数据信息
    -a —归档参数，包含了-r -d -p 参数的作用

    [root@jinc data]# cp /data/oldboy.txt /tmp/
    cp: overwrite /tmp/oldboy.txt’?
    [root@jinc data]# ls -l /tmp/
    total 4
    -rw-r–r–. 1 root root 15 Jul 5 15:19 oldboy.txt
    -rw-r–r–. 1 root root 0 Jul 5 22:33 text.txt
    -rw——-. 1 root root 0 Jul 4 15:27 yum.log

复制目录的方法–

[root@jinc data]# cp -r /data/ /tmp/
[root@jinc data]# ls -ld /tmp/data/
drwxr-xr-x. 3 root root 4096 Jul  6 01:46 /tmp/data/

说明：默认复制文件如果在目标目录中已经存在，会有覆盖提示（如何解决？？？）

1.\cp  --临时取消别名
2.unalias   --关闭别名

  1. 把/data 移动到 /root 目录下

    move 移动 ==mv 移动命令 ==剪切功能
    mv 要移动的文件或目录信息 路径
    [root@jinc data]# mv /data/ /root/

  2. 进入/root目录下的data目录，删除oldboy.txt文件
    a) 第一种方法：

    remove 移除 ==rm 移除命令
rm 要移除的文件或目录信息
[root@jinc data]# ls
oldboy.txt
[root@jinc data]# rm oldboy.txt
[root@jinc data]# ls
[root@jinc data]#

    b) 第二种方法：精确删除多个文件信息。

    第一里程：找出相应的文件数据
 find 查找数据信息命令
 find 在哪找东西 找的东西的类型 找的东西的名称
 [root@jinc ~]# find /root/data/ -type f -name "*.txt"
 /root/data/error.txt
 /root/data/right.txt
 /root/data/oldboy.txt
 第二里程：将数据进行删除
 [root@jinc ~]# find /root/data/ -type f -name "*.txt"|xargs rm -f

    c) 第三种方法：

第一里程：找出相应的文件数据
         同find 

第二里程：将数据进行删除（·· 反引号 ==$())
        [root@jinc1 data]# rm -rf 'find *.txt'

        [root@jinc1 tmp]# rm -rf $(find *.txt)

补充：管道后面的命令不识别别名

PS: 一切皆文件

反引号作用：在一个命令中嵌入执行一个命令
[root@jinc ~]# `which cat` /etc/hosts

[root@jinc ~]# $(which cat) /etc/hosts
