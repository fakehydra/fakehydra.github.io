第二关测验题目讲解

第一题：如何过滤出已知当前目录下oldboy中的所有一级目录

(提示:不包含oldboy目录下面目录的子目录及隐藏目录，即只能是第一级目录)？五种方法

创建模拟环境

cd ~
mkdir oldboy -p
cd oldboy/
mkdir ext/oldboy test xiaodong xiaofan xingfujie -p
touch jeacen oldboy wodi.gz yingsui.gz
LANG=en

方法一：利用tree命令

tree -Ld 1

方法二：利用find命令

find /root/oldboy/ -type d -maxdepth 1 （绝对路径）
find ./ -maxdepth 1 -type d （相对路径）

方法三：利用三剑客命令

ls -l /root/oldboy/|grep "d"  --以d开头
ll -F /root/oldboy/|grep "/$"  --以/结尾

ls -l /root/oldboy/|sed -n "/^d/p" --以d开头
ll -F /root/oldboy/|sed -n '/\//p' --以/结尾
ll -F /root/oldboy/|sed -n '/[/]/p' --以/结尾

ls -l /root/oldboy/|awk "/^d/" --以d开头
ll -F /root/oldboy/|awk '/\//' --以/结尾

方法四：利用硬连接

ls -l /root/oldboy/|awk '$2>1'
ls -l /root/oldboy/|awk '$2!=1'

方法五：利用*

ls -ld */

第二题：如何进行目录之间的快速切换

[root@jinc oldboy]# cd /proc/sys/net/ipv4
[root@jinc ipv4]# cd /tmp/
[root@jinc tmp]# cd -
/proc/sys/net/ipv4
[root@jinc ipv4]#

说明

-  ：还原切换以前的目录信息；等价于$OLDPWD
$OLDPWD ：记录保存切换之前的所在路径信息

cd命令用法总结

cd - :切换到上一次所在的目录信息
   ..:切换到当前目录的上一级目录
   . ：切换到当前所在的目录（一般用在移动到当前目录时使用）
   ~ ：当前登录系统的家目录信息

测验：

进入到/etc/sysconfig/network-scripts/目录，并查看你所在位置
cd /etc/sysconfig/network-scripts/
pwd

进入到上一级目录，并查看你所在位置
cd ..
pwd

进入到/root目录下面，并显示你所在位置
cd /root cd ~
pwd

快速返回到你上一次的目录位置
cd -
pwd

回到老家
cd ~ cd
pwd

第三题：一个目录中有很多文件（ls查-看时好多屏），想最快速度查看到最近更新的文件。如何看？

ls -t 
利用-t参数可以按照时间顺序排序显示文件内容（由新到旧）
ls -rt 
利用-r参数，可以按照时间信息反向排序
