mkdir

功能说明

mkdir == make directories  --创建目录；默认情况下，若要创建的目录已经存在，则会提示该目录已经存在，不会继续创建目录

语法格式

mkdir [options] [directory]
mkdir [选项] [目录]

说明：

1）注意mkdir 命令以及后面的选项和目录，每个元素之间都至少要有一个空格
2）mkdir 命令可以同时创建多个目录。

参数选项 | 解释说明

-p          |   递归创建目录（同时创建父子目录）
-m          |   设置新创建目录的默认目录对应的权限
-v          |   显示创建目录的过程

小扩展

Windows 下的目录路径样式为D:\data\test,而Linux下的路径为/data/test

使用范例

 mkdir -p 的使用

[root@jinc1 ~]# mkdir -p /oldboy/test
[root@jinc1 ~]# tree -d
.
├── 4
│   └── 5
├── oldboy
│   └── test
└── test
 mkdir批量创建目录小知识

 [root@jinc1 test]# touch ceshi.txt
 [root@jinc1 test]# cat >ceshi.txt <<E
 > 11
 > 22
 > 33
 > E
 [root@jinc1 test]# mkdir -p `cat ceshi.txt `
 [root@jinc1 test]# ll
 total 40
 drwxr-xr-x. 2 root root 4096 Jul 12 16:23 11
 drwxr-xr-x. 2 root root 4096 Jul 12 16:23 22
 -rw-r--r--. 1 root root   14 Jul 12 16:23 22.txt
 drwxr-xr-x. 2 root root 4096 Jul 12 16:23 33


 mkdir -v 参数的使用

[root@jinc1 ~]# mkdir -pv /boy/girl
mkdir: created directory `/boy'
mkdir: created directory `/boy/girl'
