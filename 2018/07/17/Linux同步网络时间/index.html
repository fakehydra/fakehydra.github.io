Linux同步网络时间

  2018-07-17    小东西

简介

这是一个小东西

很多时候，我们回因为各种不痛的需求而修改自己Linux的时间，而之后，又要自己设置回来，那样会有点小小的麻烦，所以，这时候，我们只要会这么一个小小的命令，就不用再为此而烦恼了~

安装

yum install ntpdate

使用

#  用法很简单，直接ntpdate IP 就好
#  ntpdate 210.72.145.44

date 命令

date 查看当前时间
date -s  设置时间

ntpdate 命令

ntpdate -u 210.72.145.44  --同步网络时间
注意：若不加上 -u 参数，会出现 no server found
-u :通过man ntpdate 可知 -u 参数可以越过防火墙与主机同步
210.72.145.44 ：中国国家授时中心的官方服务器

ntp常用服务器

中国国家授时中心：210.72.145.44
NTP服务器(上海) ：ntp.api.bz

美国：time.nist.gov 
复旦：ntp.fudan.edu.cn 
微软公司授时主机(美国) ：time.windows.com 
台警大授时中心(台湾)：asia.pool.ntp.org     

我在上海，所以用了一下上海的，可以正常使用
