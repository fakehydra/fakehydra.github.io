系统安装引导菜单说明

install or upgrade an exiting system --安装或更新一个存在的系统（重要）

install system with basic video driver --装系统和基本视频驱动

rescue install system  救援模式（重要）

boot from local drive 退出安装从硬盘启动
memory test 进行内存测试

系统分区说明

use all space   --使用所有空间（系统自动分区）

replace existing linux system  --替换现有的Linux系统

shrink current system   --缩小现有系统（存在一个系统，压缩原存在系统）

use free space   --使用剩余空间

creat custom layout   --创建自定义布局

常用的分区方法

/boot   200M引导分区
/swap   交换分区==内存大小的1.5倍，超过8G，只分8G即可
/       剩余所有

数据重要的情况下分区

/boot   200M引导分区
/swap   交换分区==内存大小的1.5倍，超过8G，只分8G即可
/       50-200G
4data    剩余空间都分配给数据分区

服务器使用场景不确定

/boot   200M引导分区
/swap   交换分区==内存大小的1.5倍，超过8G，只分8G即可
/       50-200G 
剩余空间不分区

虚拟软件的网络模式说明

a)可以访问外网的模式

NAT模式：优势–虚拟主机IP地址不会占用真实主机的IP地址资源
劣势–局域网其他主机不可以直接访问虚拟主机（也是可以访问的，但是比较麻烦）
桥接模式：优势–局域网其他主机可以直接访问虚拟主机
劣势–虚拟主机IP地址会占用真实主机的IP地址

b)不可以访问外网的模式

仅主机模式：只能虚拟主机直之间的交流通讯，不能访问外网

*虚拟软件的快照功能（月光宝盒）
  多做一些快照（建议）
PS:系统关闭状态制作快照会更快

必知必会题目（铺垫知识）

01.命令行提示符

[root@jinc ~]# 命令行 
root:用户信息；登陆用户
@   ：提示符中的分隔符号
jinc:主机名
~   ：显示登陆用户所在路径信息

02.书写命令格式

合法命令  空格[命令参数] 空格 [路径信息或文件数据信息]

说明：在命令中有中括号表示的信息是可有可无的，没有中括号信息是必须输入上的。

03.Linux系统目录结构

一切从跟开始

第一关测验练习题

01.创建一个目录 /data 

make directory ==mk dir==mkdir 创建目录命令

02.在/data 下面创建文件oldboy.txt

touch oldboy.txt

03.为oldboy.txt增加内容“i am studying linux ”

方法一：使用vim进行编辑

      vi/vim oldboy.txt
      i --进入编辑模式
      写入内容
      esc退出编辑模式
     ：wq 保存退出

方法二：echo 命令 --将编写内容输入到屏幕

       [root@jinc ~]# echo "hello "
                      hello <--显示到屏幕
       echo “内容” >/路径/文件
       [root@jinc ~]# echo "hello " >/data/oldboy.txt 
       [root@jinc ~]# cat /data/oldboy.txt hello 
       [root@jinc ~]# 

方法三：cat --一次性向一个文件中放入多行内容

       cat >/文件<<EOF(开始)
       内容 
       EOF（结尾）

       [root@jinc ~]# cat >/data/oldboy.txt <<EOF
       > oldboy
       > oldgirl
       > olddog 
       > EOF
       [root@jinc ~]# cat /data/oldboy.txt oldboy
       oldgirl
       olddog
