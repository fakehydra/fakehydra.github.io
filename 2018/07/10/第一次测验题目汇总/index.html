第一次测验题目汇总

Share

小测验

第一题：创建一个目录 /data

mkdir /data

第二题：在/data 下面创建一个文件 oldboy.txt

touch oldboy.txt

第三题：为oldboy.txt 增加内容为 “i am studying linux”

方法一：使用vim 编辑器
      vim oldboy.txt
      进入编辑模式：i
      输入 i am studying linux 
      退出编辑模式 ：esc
      保存退出：  ：wq

方法二：使用echo 命令
       echo "i am studying " >/data/oldboy.txt
       可以使用cat命令查看

方法三：使用cat命令
       cat >/data/oldboy.txt << E
       > i am studying linux 
       >E

第四题：把oldboy.txt 复制到 /tmp 下

cp /data/oldboy.txt /tmp

第五题：把/data 移动到 /root 目录下

mv /data/ /root/

第六题：进入 /root 目录下的 /data目录，删除oldboy.txt文件

方法一：cd /root/data 
      rm oldboy.txt

方法二：先利用find命令找到文件，然后进行删除
        find /root/data/ -type f -name "oldboy.txt" |xargs rm 

方法三：rm `find -type f -name "oldboy.txt"`
       rm $(find -type f -name "oldboy.txt")

PS:一切皆文件
   反引号作用 --在一个命令中嵌入执行一个命令

第七题：接上一题，退出到上一级目录，删除/data目录

cd ..
rm -rf /data

第八题：已知文件test.txt内容为

test
oldgirl
oldboy

请给出输出test.txt 文件内容时，不包含 oldboy 字符串的命令

创建环境：mkdir /data
         cat >/data/test.txt<<EOF
         test
         oldgirl
         oldboy
         EOF


方法一：利用grep命令进行移除
      [root@jinc ~]# grep -v "oldboy" /data/test.txt 
       test
       oldgirl

方法二：利用sed过滤
      [root@jinc ~]# sed '/oldboy/d' /data/test.txt
      test
      oldgirl

方法三：利用awk       
        [root@jinc ~]# awk '!/oldboy/' /data/test.txt 
        test
        oldgirl

awk 的另一种方法：awk 'NR!= 3' /data/test.txt

方法四：head 命令取出前几行
        [root@jinc ~]# head -2 /data/test.txt 
        test
        oldgirl

第九题：请用一条命令创建目录/oldboy/test ,即创建oldboy目录以及/test目录

mkdir -p /oldboy/test

第十题：已知/tmp下已经存在test.txt文件，如何执行命令才能把/mnt/test.txt拷贝到/tmp下覆盖掉/tmp/test.txt，而让系统不提示是否覆盖（root权限下）

第一种方法：利用\临时取消别名 
          \cp /mnt/test.txt /tmp

第二种方法：利用命令的绝对路径执行（which 命令名 -查看命令的绝对路径）
           /bin/cp /mnt/test.txt /tmp

第三种方法：|xargs 后面的命令不识别别名

第四种方法：[root@jinc1 data]# vim /root/.bashrc  
           使用 # 注释掉该别名再进行修改

ps：Linux系统自带的别名需要注释掉才能设置其他内容

第十一题：查看ett文件（共50行），查看其中20-30行内容

模拟环境创建：[root@jinc data]# seq 50 >/root/data/ett.txt 


 第一种方法：sed取行
           sed -n '20,30p' /root/data/ett.txt

 第二种方法：awk取行
           awk 'NR==20,NR==30' /root/data/ett.txt

 第三种方法：利用grep
         01. grep -A10 "20" /root/data/ett.txt   --A(after) 取20后10
         02. grep -B10 "30" /root/data/ett.txt   --B(before) 取30前10
         03. grep -C5 "25" /root/data/ett.txt    --C(center) 已25为中心取上下5

 第四种方法：head -30 /root/data/ett.txt |tail -11

 第五种方法：利用vim 显示行号：set nu

第十二题：把/oldboy目录及其子目录下所有以扩展名 .sh结尾的文件中，文件包含oldboy的字符串全部替换为oldgirl

创建模拟环境：mkdir -p /oldboy/test
             cd /oldboy
            echo "oldboy" >/oldboy/test/del.sh
            echo "oldboy" >test.sh
            echo "oldboy" >t.sh
            touch oldboy.txt
            touch alex.txt

第一种方法：find -type f -name "*.sh" |xargs sed -i 's#oldboy#oldgirl#g'
           [root@jinc1 oldboy]# find -type f -name "*.sh" |xargs sed -i 's#oldboy#oldgirl#g'
           [root@jinc1 oldboy]# cat /oldboy/test/del.sh 
           oldgirl

第二种方法：sed -i 's#oldboy#oldgirl#g' $(find -type f -name "*.sh")
           [root@jinc1 oldboy]# sed -i 's#oldboy#oldgirl#g' $(find -type f -name "*.sh")
           [root@jinc1 oldboy]# cat /oldboy/test/del.sh 
            oldgirl


第三种方法：    find -type f -name "*.sh" -exec sed -i 's#oldgirl#oldboy#g' {} \;
           [root@jinc1 oldboy]# find -type f -name "*.sh" -exec sed -i 's#oldgirl#oldboy#g' {} \;
           [root@jinc1 oldboy]# cat /oldboy/test/del.sh 
            oldboy
